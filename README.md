# Power-BI_Sales_Analysis
## Project Title
Olist E-commerce Sales Analysis using Power BI

## Objective
The goal of this project is to analyze Olist’s e-commerce sales data to identify sales trends, customer behavior, and performance metrics to support business decisions and marketing strategies.

## Tools & Technologies used
Microsoft Power BI – for dashboard creation, data modeling, and interactive visualizations
Power Query – for data cleaning and transformation
DAX (Data Analysis Expressions) – for creating calculated columns, KPIs, and measures
Excel – for initial data analysis, trend checks, and validation

## Dataset Description
The dataset was sourced from Kaggle (Olist E-commerce Dataset).
It contains detailed information about online sales transactions made on the Olist platform in Brazil.
The data includes:
Order details – order date, delivery date, order status
Product information – product category, price, and freight value
Customer details – location (city and state)
Payment information – payment type and number of installments
Customer reviews – review score and review comments
This dataset provides insights into customer purchasing behavior, delivery performance, and product sales trends.

## Insights
Sales dropped by 57% in September, then grew 15% in October and 41% in November (Black Friday).
Top 10 products showed low/no sales in September, showing strong promotional influence.
Cancellations mostly had review scores of 1, showing service issues.
Insights helped identify seasonal sales patterns and promotion planning opportunities.
